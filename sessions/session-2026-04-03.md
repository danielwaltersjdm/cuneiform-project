## Output Schema (kanesh_tablets.csv)

| Column | Description |
|---|---|
| cdli_id | CDLI P-number (e.g. P358248) |
| publication | Primary publication reference |
| authors | Author(s) |
| excavation_no | Excavation number (e.g. kt j/k 097) |
| genre | Letter / Administrative / Legal / Royal/Monumental / etc. |
| language | Akkadian / Sumerian / etc. |
| period | Old Assyrian (ca. 1950-1850 BC) |
| translation_status | no translation / uncertain / check / translator name |
| has_atf | Whether ATF body is present (4,775 tablets) |
| atf_line_count | Number of transliteration lines |
| has_translation | Whether #tr.en: lines exist (67 tablets) |
| transliteration_text | Pipe-delimited concatenation of transliteration lines |
| translation_text | Concatenated English translation lines |

## Corpus Statistics

- Total tablet records: 6,614

- With English translation: 67

## Key Finding

Of the 67 translated tablets, only 6 are Letters — the genre most relevant to behavioral analysis (price expectations, conditional clauses, merchant communications). This sharpens the translation gap problem: it is not just a quantity issue but also a genre-coverage issue.
